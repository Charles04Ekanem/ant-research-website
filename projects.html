Sio-Iso

Python script to ensure anonymity using face detection + Gaussian blur.

  • OpenCV + Haar Cascades
  • Accepts .mp4 input, Returns .avi output
  • Logs faces detected

Nsin-Uyom

Python script to inject realistic audio/visual noise into training data for realistic scenarios.

  • Simulates in background chatter into clean sound profiles
  • Supports WAV and MP3 input formats
  • Noise profiles from real-world conditions
